## Navigation Tips

### From Adventure to Secrets
1. Open the adventure file you're running
2. Check the "Northwatch Wardens Integration (DM)" section at the top
3. Follow links to relevant secret files
4. Read key secrets before the session

### From Secrets to Adventures
1. Open a secret file (e.g., Places_Secrets.md)
2. Look for "Related Adventure:" header at major location sections
3. Follow link to the adventure file
4. Review how to reveal secrets during play

### Planning Sessions
1. Start with Campaign Arc for big picture
2. Check Adventure Secrets Quick Reference (this file) for connections
3. Read specific secret sections for locations featured in today's session
4. Review adventure integration notes before running

## Player Discovery Guidelines

**The Golden Rule:** Secrets should emerge through investigation, not exposition.

**Early Campaign:**
- Strange phenomena without clear explanations
- Hints at larger patterns
- Inconsistencies that make players curious

**Mid Campaign:**
- Aeorian origin becomes clear
- Pattern of spreading effects recognized
- Multiple sites identified

**Late Campaign:**
- Full scope of Echo revealed
- Source confirmed (Salsvault)
- Nature of awakening understood
- Stakes are clear

**Order Independence:** Each adventure provides puzzle pieces. Players can discover the mystery in any sequence.
