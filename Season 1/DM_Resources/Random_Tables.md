# Random Tables

*Quick generators for improvisation*

\page

## Random Encounters (Northreach Wilderness)

**Roll 1d6 per day of travel. On a 6, roll 1d8:**

1. **2d4 Wolves** - Hungry but cautious
2. **1d6 Bandits** - Demand toll or belongings
3. **1 Owlbear** - Territorial, can be avoided
4. **Traveling Merchant** - Friendly, has basic supplies
5. **Scout from Settlement** - Has news/rumors
6. **Strange Weather** - Sudden storm, unnatural fog
7. **Magical Anomaly** - Echo manifestation
8. **NPC in Distress** - Needs help, quest hook

\page

## Weather (Northreach, Spring)

## Quest Hooks

**Roll 1d6:**

1. **Monster Attack** - Creature threatening settlement
2. **Missing Person** - NPC disappeared, foul play suspected
3. **Theft** - Valuable item stolen, need recovery
4. **Escort** - NPC needs safe passage
5. **Investigation** - Strange events, need answers
6. **Artifact** - Mysterious object discovered, study needed

## Combat Complications

**Roll 1d6 if encounter becoming boring:**

1. **Reinforcements** - More enemies arrive
2. **Terrain Hazard** - Fire, collapsing ceiling, flooding
3. **Third Party** - Another faction interrupts
4. **Environmental Change** - Weather, lighting
5. **Objective Appears** - Rescue target, McGuffin
6. **Morale Break** - Enemies flee or surrender

\page

## Use These Tables

**When:**
- Players go off-script
- Need quick content
- Improvising encounters
- Generating details

**Don't:**
- Rely on them exclusively
- Roll when not needed
- Let randomness override story

**Balance:** Use tables as inspiration, not replacement for prep
