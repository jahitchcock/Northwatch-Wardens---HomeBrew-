## üìù Notes for Contributors

### Writing Style
- **Tone**: Supportive, encouraging, practical
- **Format**: Homebrewery markdown syntax
- **Length**: Follow page guidelines in each issue
- **Voice**: Direct, actionable ("Do this" not "You might want to consider")

### Common Patterns
- Use `{{note}}` blocks for DM tips
- Use `{{descriptive}}` blocks for read-aloud text
- Use `\page` for page breaks
- Use tables for reference information
- Include examples whenever possible

### Before Submitting
- [ ] All content requirements met
- [ ] Success criteria checklist complete
- [ ] Homebrewery formatting validated
- [ ] Cross-references to other files verified
- [ ] PDF builds successfully
